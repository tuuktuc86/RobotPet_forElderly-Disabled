# RobotPet_forElderly-Disabled
A companion AI robot dog that assists with the convenience of the elderly and disabled.

## 개요
시각 장애인과 고령층, 혼자 사는 사람들을 위한 반려견 로봇입니다. 간단한 길 안내 기능과 사용자와 상호작용 요소를 지니고 있습니다.
